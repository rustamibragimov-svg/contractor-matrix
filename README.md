# Матрица контрагентов 3PL

Standalone React-приложение для управления логистическими партнёрами: договоры, тарифы, услуги и финансовые показатели по кварталам.

## Возможности

- Список контрагентов в виде таблицы или карточек
- Фильтрация по категории услуг и статусу договора
- Алерты по истекающим договорам и тарифам
- Добавление, редактирование и удаление контрагентов
- Ввод объёмов и затрат по кварталам (Q3 2025 — Q4 2026)
- Данные сохраняются в `localStorage` браузера

## Стек

- React 18 + TypeScript
- Vite
- Tailwind CSS + shadcn/ui (Radix UI)

## Запуск

```bash
npm install
npm run dev
```

Приложение откроется на `http://localhost:5173`

## Сборка

```bash
npm run build
```

## Деплой

Подключи репозиторий к [Cloudflare Pages](https://pages.cloudflare.com/) или [Vercel](https://vercel.com/) — сборка настраивается автоматически:

- **Build command:** `npm run build`
- **Output directory:** `dist`
